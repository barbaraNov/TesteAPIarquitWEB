# 🧪 Testes Unitários - Controlador de Usuários (Spring Boot)

Projeto com foco na implementação de testes unitários para um controlador de usuários utilizando **Spring Boot**, **JUnit 5** e **Mockito**.

## 🚀 Tecnologias

- Java 11+
- Spring Boot
- Spring Web & Data JPA
- MariaDB
- JUnit 5
- Mockito
